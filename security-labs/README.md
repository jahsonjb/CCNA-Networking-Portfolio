# Security Labs

This repository contains hands-on labs focused on network security fundamentals and secure infrastructure design.  
Each lab demonstrates a different layer of traffic control, visibility, and protection within enterprise networks.

## Lab Index

### Network Access Control
- **ACLs**

### Address Translation and Exposure Control
- **NAT**
  Source and destination NAT, PAT behavior, and traffic flow validation.
