# Switchport Security Labs

This repository contains hands-on labs focused on Cisco switchport security features.  
The goal is to build practical skills in securing Layer 2 environments.

## Completed Labs
- **Port Security**: Configuration and testing of switchport security commands.

## Upcoming Labs
- **DHCP Snooping**
- **Dynamic ARP Inspection (DAI)**
- **MAC Spoofing Prevention**

## Usage
Each lab includes:
- Step-by-step configuration commands
- Verification outputs
- Notes on key concepts

Clone the repo and follow along to strengthen your network security fundamentals.
